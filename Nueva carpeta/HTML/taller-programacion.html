Lógica

La lógica de programación es la base fundamental para aprender a crear soluciones mediante algoritmos. En esta etapa, el estudiante desarrolla la capacidad de analizar un problema, dividirlo en partes manejables y construir un conjunto de pasos ordenados para resolverlo. Se trabaja con conceptos esenciales como: Secuencias: instrucciones que se ejecutan en orden. Decisiones: estructuras condicionales que permiten elegir entre diferentes caminos. Repeticiones: ciclos que permiten automatizar tareas repetitivas. Variables y tipos de datos: valores que cambian durante la ejecución del algoritmo. Pensamiento computacional: descomponer, abstraer, analizar patrones y diseñar soluciones. A través de ejercicios prácticos, se impulsa al estudiante a crear sus propios algoritmos, interpretar diagramas de flujo y representar procedimientos de forma clara y precisa. Esta etapa es clave para comprender cómo “piensa” un programa y cómo convertir ideas en instrucciones correctas y eficientes.

PSeInt

PSeInt es una herramienta ideal para introducirse en la programación porque utiliza un pseudocódigo muy parecido al lenguaje natural. Gracias a esto, el estudiante puede concentrarse en aprender lógica sin preocuparse por la sintaxis compleja de un lenguaje real. En esta sección se aprende a: Declarar variables y trabajar con diferentes tipos de datos. Utilizar operadores matemáticos, lógicos y relacionales. Construir algoritmos usando SI / SINO, SEGÚN, MIENTRAS, REPETIR, y PARA. Crear y reutilizar funciones y procedimientos. Diseñar algoritmos más estructurados y limpios. Detectar y corregir errores mediante pruebas y simulaciones que ofrece PSeInt. Se realizan ejercicios crecientes en dificultad: desde programas simples como sumas o comparaciones, hasta algoritmos con ciclos anidados, validaciones de datos y cálculos más complejos. Esto prepara al estudiante para pasar al siguiente nivel: programar en un lenguaje real.

Python

Python es uno de los lenguajes de programación más usados del mundo por su simplicidad y potencia. En esta etapa, el estudiante toma los conceptos aprendidos en lógica y pseudocódigo para aplicarlos en un entorno real. Se trabaja con: Instalar y usar un entorno básico de Python. Sintaxis clara y comprensible, ideal para principiantes. Uso de variables, operadores y entrada/salida de datos. Estructuras de control: if, elif, else, bucles for y while. Manejo de listas, tuplas, diccionarios y cadenas de texto. Creación de funciones, organización del código y buenas prácticas. Introducción a la resolución de problemas reales utilizando Python. También se realizan mini-proyectos como: Calculadoras simples. Programas que procesan listas de datos. Generadores de contraseñas. Pequeños sistemas interactivos por consola. Python permite que el estudiante vea resultados reales rápidamente, lo cual hace que el aprendizaje sea dinámico y motivador.

HTML

HTML (HyperText Markup Language) es el lenguaje fundamental para crear páginas web. Permite definir la estructura y el contenido de un sitio mediante etiquetas. Con HTML se organizan textos, imágenes, enlaces, formularios, tablas y todos los elementos visuales que aparecen en una página.

En este módulo se aprende a construir la base de una web entendiendo el rol de cada etiqueta, la jerarquía del documento y las buenas prácticas de organización. También se trabaja con conceptos clave como semántica, accesibilidad, uso correcto de encabezados, secciones, listas y elementos multimedia. Es el primer paso para poder diseñar cualquier tipo de interfaz en Internet.

CSS

CSS (Cascading Style Sheets) es el lenguaje de estilos que permite dar formato, color y diseño a una página web. Con CSS se controla la apariencia de los elementos definidos en HTML: colores, tamaños, ubicación, animaciones, fuentes, márgenes y más.

Durante este módulo se aprende a utilizar selectores, clases e identificadores, así como a aplicar estilos de forma ordenada y escalable. Se estudian técnicas modernas como Flexbox y Grid para crear diseños profesionales, adaptables y responsivos.

También se trabaja en la creación de interfaces limpias y coherentes, aprendiendo sobre paletas de colores, tipografías y diseño visual. El objetivo es poder transformar una estructura simple en una página atractiva y moderna.

.box es una clase, o sea, un estilo que se le aplica solo a los elementos que tengan class="box".

JavaScript

JavaScript es el lenguaje que da vida y comportamiento a una página web. Permite crear interactividad, responder a acciones del usuario, modificar elementos en la pantalla y conectar la página con datos dinámicos.

En este módulo se aprende desde los conceptos básicos: variables, operadores, funciones, condicionales, bucles y manejo del DOM (Document Object Model). Se trabaja con eventos, animaciones simples y manipulación en tiempo real de la estructura HTML. También se introducen buenas prácticas de programación para crear scripts claros y reutilizables.

JavaScript permite pasar de una página estática a una aplicación interactiva, haciendo que botones, menús, formularios y elementos visuales funcionen de manera dinámica.
